# Privacy Notice

ARTIFACT stores scan and cleanup state locally.

## Stored Data

- Deleted artifact paths
- Artifact rule/type names
- Artifact sizes
- Project root/name when detectable
- Deletion timestamps
- Hostname in deletion metadata
- Deletion manifests for cleanup operations

## Not Stored

- File contents
- Source code contents
- Remote account credentials
- Network telemetry

## User Control

Users can delete the local database directory to clear deletion history. Users can disable file logging or remove log files from the platform data directory.
